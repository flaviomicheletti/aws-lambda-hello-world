# AWS Lambda Java

Esse é uma lambda(AWS) Java no estilo hello world.

Baseado neste tutorial https://www.tutorialspoint.com/aws_lambda/aws_lambda_function_in_java.htm



__Executando:__

Crie o arquivo`jar`.

    mvn package

Faça upload no console do AWS e execute a aplicação.
